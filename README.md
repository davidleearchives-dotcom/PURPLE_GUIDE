## What is Purple Guide?

**Purple Guide** is a real-time companion app for BTS ARMY attending the Gwanghwamun concert. It helps you navigate the venue, check crowd density, find facilities, and connect with fellow ARMY — all in one place.

---

## Features

### 🗺 Map
- Interactive Kakao Maps with venue pins
- Filter by: Toilets, Wi-Fi, Medical, Charging stations, Photo spots
- Real-time GPS tracking
- Nearby attractions (Gyeongbokgung, etc.)
- Live weather widget

### 👥 Crowd
- Real-time crowd density by zone (Exit 1, Exit 5, Merch Booth, etc.)
- ARMY-powered live reports
- Density levels: LOW → MODERATE → HIGH → VERY HIGH → EXTREME

### 💜 ARMY
- Survival Korean phrases (ko/en)
- Live group chat with fellow ARMY
- Photo spot guide
- Giveaway board

### 🚇 Transport
- Last train schedules (Subway lines 3, 5)
- Bus routes
- Taxi tips

### 🎤 Concert
- Concert timeline & schedule
- Live YouTube streams
- FAQ & rules
- Permitted / prohibited items guide

## Installation (Android APK)

1. Download [`app-release.apk`](./android/app-release.apk)
2. On your Android device: **Settings → Security → Allow unknown sources**
3. Open the downloaded APK file and tap **Install**
4. Launch **Purple Guide** 💜

> Minimum Android version: 7.0 (API 24)
